# Design System

This directory contains the foundational style guide for the project.

## Files

- `STYLE_GUIDE.md` - Comprehensive style guide with visual specifications and usage guidelines.
- `design-tokens.css` - CSS custom properties (variables) implementing the design tokens.

## Usage

1. Reference `STYLE_GUIDE.md` for design decisions.
2. Import `design-tokens.css` in your CSS to use the design tokens.
3. Follow the guidelines for consistent design across the application.

## Updating

When updating the design system, ensure both the style guide and CSS tokens are synchronized.
